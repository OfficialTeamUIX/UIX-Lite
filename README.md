# UIX-Lite
UIX Lite is a modification of the original MS Dashboard for Xbox consoles.

As such, there are plenty of issues that normally wouldn't arrise in the original dash. Since we use modified scripts in our dash, things tend to break depending on the users setup. We're hoping this GIT, using the issue tracker and tools provided will helps us coordinate fixes and help users.

To raise a bug report please fill out the following:

#UIX version number
#UIX install location (c, e, f etc)
#Xbox version (1.0, 1.1, 1.6 etc)
#Ram (64MB, or 128MB)
#Softmod or modchip (list details)
#Steps to repoduce the bug, list them in number order
#Expected behaviour
#Actual behaviour

Attach screen shots or videos.

Any other information that may help reproduce the bug.
